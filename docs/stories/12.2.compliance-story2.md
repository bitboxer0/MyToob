# Story 12.2: Hide & Blacklist Channels

## Status
Done ✅

## Story
**As a** user,
**I want** to hide content from specific YouTube channels,
**so that** I can avoid creators whose content I find inappropriate or unwanted.

## Acceptance Criteria
1. ✅ "Hide Channel" action in video context menu for YouTube videos
2. ✅ Clicking action shows confirmation dialog
3. ✅ Channel added to `ChannelBlacklist` model with channelID, reason, blockedAt
4. ✅ All videos from blacklisted channel hidden from library and search
5. ✅ "Hidden Channels" list in Settings shows all blacklisted channels
6. ✅ "Unhide" button in Settings removes channel from blacklist
7. ✅ Blacklist syncs via CloudKit (if sync enabled)
